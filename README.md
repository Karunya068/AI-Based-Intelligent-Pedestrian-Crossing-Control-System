# AI-Based-Intelligent-Pedestrian-Crossing-Control-System
## Overview
This project detects pedestrians using computer vision and intelligently controls traffic signals to improve pedestrian safety and reduce unnecessary waiting time.

## Features
- Pedestrian detection using AI
- Automatic traffic signal control
- Real-time video processing
- Improved road safety

## Technologies Used
- Python
- OpenCV
- YOLO
- NumPy

## Project Structure
main.py
requirements.txt
README.md

## How to Run
1. Install Python
2. Install dependencies
pip install -r requirements.txt
3. Run
python main.py
