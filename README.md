# AI-Based-Intelligent-Pedestrian-Crossing-Control-System
Developed an AI-enabled pedestrian crossing system using Python, YOLOv8, OpenCV, and Arduino to improve road safety through real-time pedestrian detection and adaptive traffic signal control.
**What Was the Outcome?**
The system successfully detected pedestrians in real time and dynamically controlled the traffic signal. It extended the red signal whenever pedestrians were still crossing, prevented new pedestrians from entering using servo-controlled gates, and improved overall pedestrian safety. The project demonstrates the practical application of AI and embedded systems in intelligent traffic management for smart cities.
